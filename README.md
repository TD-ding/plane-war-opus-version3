# 飞机大战 · Plane War

一个使用 **HTML5 Canvas** 实现的单文件飞机大战小游戏。纯前端、零依赖，浏览器直接打开 `index.html` 即可游玩。

## 简介

经典纵版射击游戏：操控己方战机躲避并击落不断下落的敌机。击中敌机得分，被敌机撞击则扣除生命值，生命值耗尽即游戏结束。随时间推移敌机会变得更快、更密集，难度逐步提升。

## 功能特性

- 🎮 键盘（方向键 / WASD）或触屏拖动控制飞机移动
- 🔫 自动连续射击
- 👾 敌机从顶部生成、下落并轻微摆动；大型敌机血厚、得分更高
- 💥 子弹击中与撞击的粒子爆炸特效
- ❤️ 生命值系统（5 点血）、实时得分与历史最高分（localStorage 持久化）
- 🌌 滚动星空背景
- 📈 随时间递增的难度
- 🏁 开始界面、游戏结束界面与一键重新开始
- 🛡️ 主循环错误兜底、dt 钳制、localStorage 容错等基础错误处理

## 技术栈

- HTML5 Canvas 2D
- 原生 JavaScript（ES6，无任何外部库 / 框架）
- CSS3
- localStorage（最高分存储）

## 如何运行

方式一：直接双击打开 `index.html`，用任意现代浏览器（Chrome / Edge / Firefox / Safari）即可游玩。

方式二（推荐，避免个别浏览器本地文件限制）：在项目目录启动一个本地静态服务器：

```bash
# Python 3
python3 -m http.server 8000
# 然后浏览器访问 http://localhost:8000
```

## 操作说明

| 操作 | 按键 |
| --- | --- |
| 移动 | 方向键 / `W` `A` `S` `D`，或触屏拖动 |
| 射击 | 自动开火 |
| 开始 / 重新开始 | `回车` 或 点击屏幕 |
