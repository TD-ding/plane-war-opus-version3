# 飞机大战 · Plane War

一个使用 **HTML5 Canvas** 实现的单文件飞机大战小游戏。纯前端、零依赖，浏览器直接打开 `index.html` 即可游玩。

## 简介

经典纵版射击游戏：操控己方战机躲避并击落不断下落的敌机。击中敌机得分，被敌机撞击或被敌弹击中则扣除生命值，生命值耗尽即游戏结束。连续击毁敌机可累积**连击**让得分翻倍；火力会随表现成长，打得好时子弹从单发升级为多发散射；击毁敌机有概率掉落道具（护盾、回血、强化火力、清屏炸弹）。随时间推移敌机会变得更快、更密集，大型敌机还会朝你开火，难度逐步提升。

## 功能特性

- 🎮 键盘（方向键 / WASD）或触屏拖动控制飞机移动
- 🔫 自动连续射击
- 🔥 **火力成长系统**：连击/拾取道具时子弹由单发升级为 2/3/4 发散射，被撞则降一级
- ⚡ **连击系统**：连续击毁敌机累积连击，达到档位后得分翻倍（最高 x4），HUD 实时显示连击数与倍率
- 🎁 **道具掉落**：敌机击毁后偶尔掉落，飞过去拾取，过一会儿没捡到会闪烁并消失
  - 🛡️ 护盾：一段时间内被撞 / 被打不掉血
  - ➕ 回血：恢复一点生命值
  - 🔺 强化火力：火力等级 +1
  - ✸ 清屏炸弹：拾取后存入库存，紧急时按 `B` 清空全屏敌机与敌弹
- 👾 敌机从顶部生成、下落并轻微摆动；大型敌机血厚、得分更高，**并会朝玩家方向射击子弹**，需要躲避
- 🔊 **程序化音效**：射击、命中、爆炸、被撞、拾取、炸弹等电子音效，全部用 WebAudio 实时合成，无任何外部音频文件，可按 `M` 静音
- 💥 子弹击中与撞击的粒子爆炸特效、屏幕震动、得分飘字
- ❤️ 生命值系统（5 点血）、实时得分与历史最高分（localStorage 持久化）
- 🌌 滚动星空背景
- 📈 随时间递增的难度
- 🏁 开始界面、游戏结束界面与一键重新开始
- 🛡️ 主循环错误兜底、dt 钳制、localStorage 容错、音频上下文容错等基础错误处理

## 技术栈

- HTML5 Canvas 2D
- 原生 JavaScript（ES6，无任何外部库 / 框架）
- CSS3
- localStorage（最高分存储）

## 如何运行

方式一：直接双击打开 `index.html`，用任意现代浏览器（Chrome / Edge / Firefox / Safari）即可游玩。

方式二（推荐，避免个别浏览器本地文件限制）：在项目目录启动一个本地静态服务器：

```bash
# Python 3
python3 -m http.server 8000
# 然后浏览器访问 http://localhost:8000
```

## 操作说明

| 操作 | 按键 |
| --- | --- |
| 移动 | 方向键 / `W` `A` `S` `D`，或触屏拖动 |
| 射击 | 自动开火（按住 `空格` 加速连射） |
| 释放清屏炸弹 | `B`（需库存 > 0） |
| 暂停 / 继续 | `P` 或 `Esc` |
| 静音 / 取消静音 | `M` |
| 开始 / 重新开始 | `回车` 或 点击屏幕 |

## 玩法提示

- **连击**：连续击毁敌机会累积连击数，达到 5/10/20 连击时得分倍率分别提升到 x2/x3/x4；约 2.2 秒内没有新的击毁，或被击中，连击就会清零。
- **火力成长**：拾取“强化火力”道具会让火力等级 +1，子弹随之变成 2/3/4 发散射；被撞或被敌弹击中会降一级，所以要尽量别挨打。
- **道具**：敌机击毁后有概率掉落道具并向下飘落，飞过去碰到即可拾取；道具存在约 7 秒，临近消失会闪烁提醒。
- **清屏炸弹**：拾取后存入库存（最多 3 个），被敌机包围时按 `B` 一键清空全屏敌机与敌弹并计分。
- **躲敌弹**：大型敌机会朝你当前位置发射粉色子弹，记得走位躲避；护盾期间可免疫一切碰撞与敌弹伤害。
